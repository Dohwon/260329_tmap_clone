# Strategy Roadmap

## 장기 방향

### 1. 데모형 MVP에서 실주행형 MVP로 전환

- 경로/검색/근처 POI fallback을 점진적으로 제거
- 실시간 교통과 단속 데이터를 신뢰 가능한 외부 소스로 연결
- 합류 옵션을 heuristic이 아닌 실제 재탐색 결과 기반으로 전환

### 2. 티맵 수준 체감 확보

- 고속도로·국도 전체 노선별 속도/정체/카메라 갱신 주기 설계
- 도로 선택이 실제 경로 탐색 엔진의 강한 선호도로 반영되게 개선
- GPS 이동, follow 모드, 권한 실패 상태를 모바일 기준으로 다듬기

### 3. 운영 가시성

- 실데이터/시뮬레이션 상태 배지 추가
- GPS 실패/권한 거부/프록시 실패 진단 UI 추가
- QA 체크리스트를 실기기 시나리오 중심으로 고정화
- dev/prod 모두 동일한 TMAP 키 로딩과 프록시 진단 흐름 유지
